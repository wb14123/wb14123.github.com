Using Cinnamon Currently

Posted on 30 Nov 2012, in workspace, tagged linux, desktop, and UI

After Ubuntu start to use Unity as its default desktop, I choose to Archlinux, use openbox with tint2 and Gnome 3 for a long time. Both of them have some features I’d like. When I installed Linux Mint for a friend some days ago, I get to know META and Cinnamon. META is a fork of Gnome 2 and make me feel like home. After such a long time when I reuse a Gnome 2 desktop, I realize what makes me like Linux at the first time: the ability to configure such a lot things. As I’m new to Linux at that time, the configures are mainly through GUI. Gnome 2 gave me a chance.

But re-thought the love of Gnome 2 today, it is not the Unix philosophy: “Do one thing and do it well”. It provides a windows manager, a panel, power manager, key-bindings and so on. Since the GUI program use text to communicate as command line tools, there is no need to split them. Put them all in one box is more stable. I think there are two important points for desktop environment:

  • Ability and flexible to configure.
  • Give the user an out-of-box and stable environment.
